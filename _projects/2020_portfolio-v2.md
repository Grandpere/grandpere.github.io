La v1 était du HTML et CSS statique or c'était contraignant de devoir modifier à la main à chaque fois.

La v2 utilise un fichier json contenant les données qui seront lues par le js et qui génèrera mon contenu. Je n'aurai plus qu'à ajouter des données au json en respectant le format que j'ai choisi.
